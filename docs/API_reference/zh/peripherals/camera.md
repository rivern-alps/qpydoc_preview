# camera  - 摄像头

该模块包含了摄像头预览，照相机，扫码功能。

> 目前支持型号：EC600N系列、EC800N系列、EC600M系列、EC800M系列、EG810M系列、EC600U-CN系列、EC200U-CN系列。
>
> 如果需要预览，在初始化camera对象之前，请先参照[machine模块的lcd类的内容](machine.LCD.md)正确初始化LCD对象。
